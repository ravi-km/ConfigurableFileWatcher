# ConfigurableFileWatcher
ConfigurableFileWatcher

Configurable File watcher with caching mechanism.

Configurations:<br/>
  *File path to watch<br/>
  *File name to watch<br/>
  *Destination to move the file to, once found<br/>
  *Cache Timeout<br/>
  *Max retries for cache<br/>
  *Polling Interval<br/>
